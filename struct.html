Движение¶

Открывает новое окно со списком движения на предприятии:

Здесь содержится необходимая табличная информация о движении на предприятии, а также кнопки:

Добавить:открывает форму для добавления нового движения в таблицу.
Редактировать:открывает форму для редактирования выбранного движения (необходимо выделить строку нажатием левой кнопки мыши).
Удалить:удаляет движение из таблицы (также необходимо выделить строку левой кнопкой мыши).

Справочники ‣ Сотрудники¶

Открывает новое окно с сотрудниками. Это одна из характеристик, присваиваемых во время добавления/редактирования движения.

Справочники ‣ Должности¶

Открывает справочник со сведениями о должностях.

Справочники ‣ Категории¶

Открывает справочник со сведениями о категориях:

Добавить:открывает форму для добавления новой категории в таблицу.
Редактировать:открывает форму для редактирования выбранной категории (необходимо выделить строку нажатием левой кнопки мыши).
Удалить:удаляет категорию из таблицы (также необходимо выделить строку левой кнопкой мыши).
Закрыть:закрывает окно категории.

Справочники ‣ Изделия¶

Открывает окно для добавления нового изделия в таблицу.

Справочники ‣ Лаборатория¶

Открывает окно для добавления новой лаборитории в таблицу:

Добавить:открывает окно для добавления новой лаборатории в таблицу.
Редактировать:открывает форму для редактирования выбранной лаборатории (необходимо выделить строку нажатием левой кнопки мыши).
Удалить:удаляет лабораторию из таблицы (также необходимо выделить строку левой кнопкой мыши).
Закрыть:закрывает окно лаборатории.
